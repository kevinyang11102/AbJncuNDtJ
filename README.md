# 学生成绩管理系统 python158

学生成绩管理系统是一款基于Web的信息管理系统，主要用于处理学生的成绩信息和选课管理。本项目采用Python语言开发，以Flask框架为基础搭建。

## 技术栈

- 后端：Python 3.x
- Web框架：Flask
- 数据库：SQLAlchemy

## 功能模块

- **用户管理**
  - 登录/登出
  - 用户权限控制
- **学生信息管理**
  - 学生信息录入
  - 学生信息查询
  - 学生信息修改
- **课程管理**
  - 课程信息录入
  - 课程信息查询
  - 课程信息修改
- **选课管理**
  - 学生选课
  - 选课结果查询
  - 成绩录入与查询

## 系统角色

- 管理员：负责用户管理、学生信息管理、课程管理等功能模块的操作。
- 教师：负责成绩录入与查询、课程信息查看。
- 学生：负责选课、查询选课结果和成绩。

## 运行环境

- 操作系统：Linux/Windows
- Python版本：3.x
- 依赖管理：pip

## 部署步骤

1. 安装Python环境。
2. 使用pip安装项目所需依赖。
3. 设置数据库环境。
4. 运行Flask应用。
